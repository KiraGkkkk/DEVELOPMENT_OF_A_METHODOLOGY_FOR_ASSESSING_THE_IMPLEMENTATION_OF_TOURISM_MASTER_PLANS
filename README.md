Автор: Коржова И.Н.
Тема: Разработка методики оценки реализации туристических мастер-планов (на примере территорий России)
Год: 2026 (актуальные данные до 2025 г.)
Направление подготовки: 07.04.04 Градостроительство
Разработана методика оценки туристических мастер-планов России, объединяющая анализ экономики (DEA), экологии (NDBI), связности, сегрегации и согласованности документов. Апробирована на Дальнем Востоке, Кубани и Алтае. Решает проблему отсутствия системного мониторинга 42 схем для 12 макротерриторий.
